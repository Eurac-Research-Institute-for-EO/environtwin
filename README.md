# PLANETSCOPE DATA PROCESSING

## About
This repository contains all processing steps developed during the EnvironTwin project. From ordering and downloading images, to geometric correction and the calculation of NDVI time series from daily PlanetScope data. 

Additionally there are some old scripts and tests located in the respective folders that can be used and modified. 

### Useful links
PlanetScope data informnation [PlanetHub](https://docs.planet.com/data/imagery/).

APIS for ordering and downloading data [API](https://docs.planet.com/develop/apis/).

Some useful examples and Jupyternotebooks [Guide](https://docs.planet.com/guides/).
